Project One

DreamFlix

DreamFlix is a Netflix-inspired video streaming platform built with React on the frontend and Node.js/Express on the backend. It features dynamic movie browsing, user authentication, and personalized content filtering. Tech Highlights: React.js: Built with component-based architecture for responsive UI, routing with React Router, and state management using Context API or Redux. Node.js & Express: RESTful API for handling user auth (JWT), movie data CRUD, and secure backend logic. MongoDB (optional): Used as a NoSQL database to store user profiles, watchlists, and content metadata. JWT & bcrypt: Secure login with encrypted tokens and hashed passwords. Responsive Design: Mobile-friendly layout styled with CSS Modules or Tailwind.

    Key Features:
  • User registration and login
  • Browse, search, and filter movies/shows
  • Personalized watchlists
